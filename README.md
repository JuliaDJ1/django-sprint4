# django_sprint4
Это блог-платформа на Django, где пользователи могут создавать посты, комментировать, редактировать профили и т.д. Реализована аутентификация, пагинация, изображения в постах, комментарии и кастомные страницы ошибок.
Запуск
manage.py runserver
Перейдите на http://127.0.0.1:8000/.
